# Maze Solver - Enhanced Version

Modern hexagonal maze solver with beautiful UI.

## Quick Start

```bash
# Build
mvn clean package

# Run
./run.sh
# or
java -jar target/maze-solver-2.0.0.jar
```

## Features

- Modern dark/light theme
- Hexagonal grid with Dijkstra's algorithm
- Interactive maze creation
- Import/Export functionality
- Keyboard shortcuts (Ctrl+O, Ctrl+S, Ctrl+L, Ctrl+R, F1)
- Dynamic zoom (5-30 dimensions)

## Build Requirements

- Java 21+
- Maven 3.6+
